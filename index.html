📝 Le Prétérit

Construction du Prétérit en anglais

✅ Forme affirmative

Sujet + V-ed (réguliers) / forme irrégulière + complément

She worked yesterday. (régulier)

He went to school. (irrégulier : go → went)

They had a meeting. (irrégulier : have → had)

❌ Forme négative

Sujet + did not (didn't) + base verbale + complément

She didn't work yesterday.  |  He didn't go to school.

⚠️ On utilise la base verbale après didn't, pas le prétérit !

❓ Forme interrogative

Did + sujet + base verbale + complément + ?

Did she work yesterday?  |  Did he go to school?

💡 Did est invariable — il ne change pas selon le sujet.

🔁 Exemples de verbes irréguliers

go → went → gone
be → was/were → been
have → had → had
do → did → done
say → said → said
get → got → gotten
